# Task Manager Backend Reference

# Requirements

* Ruby >= 2.4.0

* Rails >= 5.0

* PostgreSQL >= 9.3.13


# Quick Start

* git clone git@github.com:sreerajsomarajan/task_manager.git

* cd task_manager

* Change PGSQL_USERNAME and PGSQL_PASSWORD in database.yml.example and save as database.yml

* rails db:setup

* rails s
